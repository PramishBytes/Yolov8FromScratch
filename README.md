# 🎯 YOLOv8 From Scratch

## 📌 About

This repository will contain a step-by-step implementation of the YOLOv8 (You Only Look Once v8) object detection algorithm from scratch using Python.

It is being built to:

- Understand the fundamentals of object detection  
- Learn how YOLOv8 works internally  
- Implement core components from scratch  
- Explore deep learning and computer vision concepts
